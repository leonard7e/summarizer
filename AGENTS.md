## Grundlegende Richtlinien
### Sprache
Der generierte Code sollte auf Englisch geschrieben sein. Ebenso sollte die Dokumentation auf Englisch sein.

## Code Qualität
### Fasslichkeit der Code-Generierung für Menschen
* Vermeide lange Prozeduren. Spalte sie in mehrere Prozeduren auf.
* Vermeide es, duplizierten Codes zu generieren. 
* Nutze Methoden (Traits wie Result | Option | Iterator). Vermeide verschachtelte Ausdrücke wie if | match | for, wenn diese die Lesbarkeit verringern.
