[[quickstart]]
== Quick Start

This section gets you from zero to your first summary in five minutes, using a locally running Ollama instance (no API key required).

=== Step 1: Install Ollama and pull a model

[source,bash]
----
# Install Ollama (see https://ollama.com for other platforms)
curl -fsSL https://ollama.com/install.sh | sh

# Pull a model
ollama pull llama3
----

=== Step 2: Configure summarizer

Run the interactive setup wizard:

[source,bash]
----
summarizer init
----

The wizard will ask for:

* OpenRouter API key — press Enter to skip
* Gemini API key — press Enter to skip
* Ollama base_url — accept the default `http://localhost:11434`
* Default model — select `ollama:llama3` from the list

The configuration is saved to `~/.config/summarizer/config.yaml`.

=== Step 3: Summarize a file

[source,bash]
----
summarizer report.txt
----

The summary is printed to standard output.

=== Step 4: Try a custom prompt

[source,bash]
----
summarizer -p "Extract all action items as a numbered list." meeting_notes.txt
----

=== Step 5: Summarize multiple files

`summarizer` accepts any number of files. Files that exceed the context window are processed in batches automatically:

[source,bash]
----
summarizer chapter1.txt chapter2.txt chapter3.txt chapter4.txt
----

Progress is printed to standard error so you can pipe the final output cleanly:

[source,bash]
----
summarizer *.txt > summary.txt
----
